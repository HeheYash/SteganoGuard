# Screenshots

Add screenshots here and reference them in the main README, e.g.:

```
screenshots/
├── hide-tab.png       # The "Hide Message" tab with an image loaded
├── extract-tab.png    # The "Extract Message" tab showing a decoded message
└── dark-mode.png       # Dark mode view
```

Recommended way to capture these:
1. Run the app locally (`python3 -m http.server 8000`).
2. Use your browser's screenshot tool, or a full-page capture extension, at a standard width (e.g. 1440px) for consistency.
3. Save as PNG into this folder, then reference them from `README.md`:

```markdown
![Hide Message tab](screenshots/hide-tab.png)
```
